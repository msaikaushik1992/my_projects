My Movies Documentation

Project Limitations:

Complex page like movie details page sometimes loads very slowly. The size of thumbnails fetched from API is variable and depending upon that sometimes it takes seconds to load a particular page properly.

I have reduced the display of number of items like cast information and movies information being fethed from API for performance related matters so that the user doesn't have to wait much longer.

There are some lines of codes which are similar for a number of pages with a very minute difference. This code must be organised properly into individual files and more task should be delegated to helper functions.

Rotten Tomatoes API some times fails to load the data. However this happens rarely.

Extentions to Project:

Automatic google like search suggestions can be provided to the user, when the user tries to search for a movie based on the characters that he/she inputs.

Complete information about the cast and crew members can be displayed to user whenever the thumbnail or name of any of the member is clicked
