Details Page

Details page is the most complex page of my project.

The complete page is divided into four different sections:

  • Title
  • Plot
  • Rating
  • Cast

To fetch the entire details about the movie , Rotten tomatoes API is used. Movie details like movie poster, movie plot, movie rating and information about cast and crew are displayed

Whenever the users clicks on a movie on any of the page(except COMING SOON page, which only displays movies which are coming soon), the user gets redirected to Moviedetails page (Referred from Rotten tomatoes API ) and the ID of that movie gets appended to the url. Now on the details page I am extracting the ID for that movie from URL to populate the results by making API requests.

The Title section includes the Poster of movie along with title itself

The Plot section includes the plot of the movie

The Rating section includes the rating of the movie

The last section includes the information of the cast in the form of actors names and their images

The jQuery file for Details page is

MovieDetails.js.

It contains all the jQuery code to fetch the details about the movie dynamically.
